

# Student Management System:

The Student Management System is a software application designed to handle student-related tasks, including enrollment, course management, grading, and student information tracking. This system aims to simplify administrative processes within educational institutions.

## Features:
### Student Enrollment: 
Register new students into the system, including personal information and contact details.
### Course Management: 
Add, edit, or remove courses offered by the institution.
### Grading System: 
Record and manage student grades for various courses.
### Student Information:
Store and update student details such as academic performance, attendance, and disciplinary records.
